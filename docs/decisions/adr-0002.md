# ADR-0002 — Ownership and identity

## Decision
Each artifact owner links to the contributor's git user identity. The system does not lean heavily on CODEOWNERS. See requirement WP-008.

## Context
Promoted from decision D-002 in the IBuildOS master requirements specification (2026-06-30).

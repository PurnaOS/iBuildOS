# ADR-0001 — Implementation language and distribution

## Decision
The implementation language is TypeScript, compiled to a single self-contained executable with Bun. The same stack also powers the local UI app (see ADR-0004). This assumes TypeScript integrates cleanly with the coding harness, which should be validated early. See section 10 of the master spec.

## Context
Promoted from decision D-001 in the IBuildOS master requirements specification (2026-06-30).

# ADR-0007 — Parallel-agent conflict resolution

## Decision
Conflicts are finalized locally by a developer before merge. IBuildOS surfaces conflicts and validation state but never auto-resolves. See requirement VC-009.

## Context
Promoted from decision D-007 in the IBuildOS master requirements specification (2026-06-30).

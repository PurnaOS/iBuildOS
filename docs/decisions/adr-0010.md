# ADR-0010 — Staleness detection

## Decision
Use the team's existing, already-built staleness checker. IBuildOS integrates and runs it rather than inventing a heuristic. See requirements CQ-005 and AG-004.

## Context
Promoted from decision D-010 in the IBuildOS master requirements specification (2026-06-30).

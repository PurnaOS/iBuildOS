# ADR-0006 — Status and workflow customization

## Decision
Statuses and workflows are defined in the data model and artifact taxonomy (the type profile, section 9 of the master spec), not in a separate config surface. See requirements TS-003, TS-004, and WP-003.

## Context
Promoted from decision D-006 in the IBuildOS master requirements specification (2026-06-30).

# ADR-0004 — UI build versus borrow

## Decision
Both, by role. A local, conductor-like working app (UI-006) for daily use, plus Backstage or MkDocs for a hosted, read-only docs site (UI-009) so stakeholders can browse.

## Context
Promoted from decision D-004 in the IBuildOS master requirements specification (2026-06-30).

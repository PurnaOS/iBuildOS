# Release

A **Release** (or Version) is a shippable cut. Work items declare `planned_for` a
release on [BacklogItem](backlog-item.md), so a release's contents are derived
from the graph rather than maintained as a separate list. Release notes can be
generated by walking everything `planned_for` it.

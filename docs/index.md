# iBuildOS Knowledge Bundle

The single source of truth for the iBuildOS project's software lifecycle, stored
as OKF documents and validated by iBuild. The bundle root and artifact globs are
configured in `.ibuildos.yaml` at the repository root.

# Contents

* [Types](types/) - the SDLC profile: artifact type definitions
* [Requirements](requirements/) - statements of need
* [Work](work/) - epics and tasks
* [Tests](tests/) - checks that verify requirements

The system shall provide **batch migration operations** — bulk import of incumbent issues/tests into typed artifacts, bulk owner/`User` backfill, agent-assisted bulk classification of legacy docs to OKF types via a reviewable mapping table, and verify-link inference for existing tests — applied idempotently and delivered as reviewable proposals (`IN-005`), so a large back-catalog migrates in vetted batches rather than one artifact at a time.

---
type: FunctionalRequirement
id: FR-0001
title: Deterministic OKF frontmatter parsing with line-accurate findings
owner: srini
status: accepted
priority: must
---

The engine shall parse OKF frontmatter (a `--- ... ---` YAML block + markdown
body) deterministically, mapping every value to its 1-based source line so
findings point at the exact line, and tolerating prose/malformed files without
rejecting the bundle.

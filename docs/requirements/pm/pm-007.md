The system shall aggregate, for the current `User` (resolved from git identity), everything on their plate — work assigned to them, reviews requested of them, bugs they own, and pending notifications — exposed via the **CLI** (e.g., `iBuild mine`) and the UI (`UI-015`). Derived from assignment + review state, not hand-maintained.

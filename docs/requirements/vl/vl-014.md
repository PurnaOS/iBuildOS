The CI gate (`VL-008`) shall support a **report-only / non-blocking** mode that annotates findings without failing the build, shall compose with (never require replacing) an existing pipeline, and shall document a promotion path from non-blocking to blocking as the baseline (`VL-013`) shrinks — so adopting the gate on a brownfield repo does not halt in-flight feature work.

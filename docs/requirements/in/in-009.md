Init shall configure a ready-to-use default **workflow**: the process loop (spec → plan → tasks → implement → validate → review → release) with the profile's default statuses/lifecycles, plus wired-in CI validation (`VL-008`) and a pre-commit hook (`VL-009`). All of it stays customizable in the profile afterward (decision D-006).

Every artifact's `owner` (and assignee) shall resolve to a **`User`** entity (`WP-009`) that maps to a git user identity (name/email/handle), so artifacts and changes attribute to real people via git history. The system shall not depend heavily on `CODEOWNERS` for ownership. (decision D-002)

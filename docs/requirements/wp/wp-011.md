The shipped base profile shall include optional team-memory artifact types (e.g., `MeetingNote`, `RetroAction`, `StandupLog`) so standups, retros, and decisions are captured as linkable, version-controlled knowledge rather than living only in external chat; teams may disable them. (surfaced in the UI via `UI-016`)
